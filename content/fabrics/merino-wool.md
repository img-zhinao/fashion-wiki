# 美利奴羊毛 — 高端天然纤维之王

> 美利奴羊毛（Merino Wool）产自美利奴绵羊，纤维直径仅 15-24 微米，比人类头发还细，具有天然温控、抗菌除臭、吸湿排汗等卓越性能，是高端服装和户外装备的首选面料。

## 核心性能优势

### 1. 天然温控（冬暖夏凉）
- **原理**：羊毛纤维卷曲结构形成空气层，隔热保温
- **数据**：冬季保暖性比棉高 30%，夏季透气性与棉相当
- **场景**：一件美利奴羊毛 T 恤可以从春天穿到秋天

### 2. 吸湿排汗
- **原理**：羊毛纤维可吸收自身重量 35% 的水分而不感觉湿
- **数据**：排汗速度比棉快 2 倍，比涤纶快 1.5 倍
- **场景**：运动、徒步、骑行时保持干爽

### 3. 抗菌除臭
- **原理**：羊毛纤维表面有天然鳞片结构，抑制细菌生长
- **数据**：穿着 7 天不洗，异味比棉 T 恤少 80%
- **场景**：旅行、户外、差旅（减少行李重量）

### 4. 抗紫外线
- **原理**：羊毛纤维天然吸收 UV 辐射
- **数据**：UPF 值 40+，相当于防晒衣
- **场景**：户外、高海拔、热带地区

### 5. 防火阻燃
- **原理**：羊毛纤维含氮和水分，不易燃烧
- **数据**：离开火源后自熄，不熔化、不滴落
- **场景**：消防、 military、航空航天

| 层级 | 功能 | 美利奴羊毛应用 |
|------|------|---------------|
| **基础层（Base Layer）** | 贴身排汗 | 美利奴羊毛内衣、T 恤 |
| **中间层（Mid Layer）** | 保温隔热 | 美利奴羊毛抓绒、毛衣 |
| **外层（Outer Layer）** | 防风防水 | 美利奴羊毛混纺外套 |

- **高端 T 恤**：比棉 T 恤更透气、更防臭、更易打理

## 护理方法

| 护理项目 | 建议 |
|---------|------|
| **洗涤** | 冷水或 30℃ 以下温水，羊毛专用洗涤剂 |
| **机洗** | 可机洗（选羊毛/轻柔模式），装洗衣袋 |
| **干燥** | 平铺阴干，避免悬挂（防止变形） |
| **熨烫** | 低温熨烫，垫布保护 |
| **收纳** | 折叠存放，避免重压；放樟脑丸防虫 |
| **注意** | 避免接触尖锐物品，湿态时勿拉扯 |
