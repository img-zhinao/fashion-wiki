# 天丝（莱赛尔）— 环保面料百科

> 天丝（TENCEL™）是奥地利兰精公司生产的莱赛尔纤维，以木浆为原料，生产过程环保闭环，面料具有丝绸般光泽、棉的透气性和优异的垂感。

## 天丝的核心优势

### 1. 极致透气（比棉更透气）
- 纤维结构微孔丰富，吸湿性是棉的 1.5 倍
- 夏季穿着体感温度可降低 2-3℃
- 适合：夏季衬衫、连衣裙、睡衣、床品

### 2. 丝绸般光泽与垂感
- 表面光滑，光线反射均匀，呈现高级哑光光泽
- 垂坠感接近真丝，但比真丝更易打理
- 适合：高端衬衫、飘逸长裙、丝巾

### 3. 环保闭环生产
- 溶剂回收率 99%，几乎零排放
- 原料来自可持续林场，不消耗耕地
- 对比 cotton：用水量减少 95%
- 对比 polyester：无石油依赖，可生物降解

### 4. 亲肤低敏
- 纤维表面光滑，摩擦系数低
- 不易滋生细菌，适合敏感肌
- 适合：婴幼儿服装、贴身内衣、医用纺织品

## 天丝的护理方法

| 护理项目 | 建议 |
|---------|------|
| **洗涤** | 冷水或 30℃ 以下温水，手洗或轻柔机洗 |
| **洗涤剂** | 中性洗涤剂，避免含酶洗衣粉 |
| **脱水** | 避免高速甩干，平铺晾干 |
| **晾晒** | 避免暴晒，阴干最佳 |
| **熨烫** | 中低温熨烫，可趁微湿时熨 |
| **收纳** | 避免重压，防止褶皱变形 |
| **注意** | 避免接触尖锐物品，天丝湿态强度降低 |

## 天丝的常见问答

**Q: 天丝和莫代尔有什么区别？**
A: 同属再生纤维素纤维，但天丝（莱赛尔）生产工艺更环保（闭环溶剂法），纤维强度更高，光泽和垂感更优。莫代尔更柔软但强度较低，易变形。

**Q: 天丝会起球吗？**
A: 相比棉和羊毛，天丝抗起球性较好。但长期摩擦（如腋下、臀部）仍可能轻微起球，建议避免与粗糙表面摩擦。

**Q: 天丝适合什么季节？**
A: 最适合春夏，但全年可用。夏季凉爽透气，春秋可制作稍厚款或与其他纤维混纺。

**Q: 为什么天丝比棉贵？**
A: 生产工艺更复杂（闭环溶剂法设备投资高），环保认证成本高，且原料来自可持续林场。但性价比高于真丝，是高端环保面料的平民化选择。

**Q: 天丝会缩水吗？**
A: 首次洗涤可能有 3-5% 缩水，建议购买时预留。之后缩率稳定，遵循护理指南可最大程度减少缩水。
